My own CSS cheatsheet

CSS Visual Rules

Name Example Description
font-size font-size: 30px; The font-size CSS property is used to set text sizes.
Background color background-color: red; The background-color CSS property controls the background color of elements.
Font weight font-weight: bold; The font-weight CSS property can be used to set the weight (boldness) of text.

CSS Colors

Type Example Description
Name keywords color: grey; Basic color name keywords to set a value in CSS
Alpha values rgba() / hsla(); Alpha values determine the transparency of colors in CSS.
Hexadecimal colors color: #808080; CSS colors can be represented in hexadecimal (or hex) notation.
HSL Colors background-color: hsl(200, 70%, 50%); Colors can be declared with the HSL color system using hsl() syntax
rgb() colors background-color: rgb(200, 150, 50); RGB() should be supplied with three values representing red, green, and blue. These values range can from 0 to 255
